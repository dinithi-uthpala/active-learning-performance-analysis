# Active Learning and Student Performance Analysis

## Project Overview
This project analyzes whether active learning engagement improves student academic performance. It was completed as part of the Theory and Practices in Statistical Modelling module at SLIIT.

## Objective
The main objective of this project was to examine the relationship between student engagement and academic performance using data analysis and statistical modelling techniques.

## Tools and Technologies
- R
- RStudio
- Data Cleaning
- Feature Engineering
- Descriptive Analytics
- Welch Two Sample t-test
- Linear Regression

## Methodology
- Prepared and cleaned the dataset for analysis.
- Created an active ratio to measure student engagement.
- Divided students into lower and higher engagement groups.
- Compared performance between groups using statistical testing.
- Built a regression model to evaluate how engagement predicts academic performance.

## Key Findings
- Higher engagement students scored approximately 6.5 marks more on average.
- The difference was statistically significant.
- Engagement showed a positive relationship with academic performance.

- Exploratory data analysis

- Regression modelling
- Interpretation of analytical results
